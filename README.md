# 🚗 Performance Garage

A modern React-based supercar dashboard that showcases 15+ high-performance cars with detailed specifications in a clean and responsive UI.

---

## ✨ Features

* 🚗 15+ Supercars with specifications (speed, horsepower, description)
* ⚡ Dynamic data rendering using React
* 🧩 Reusable component-based architecture
* 🎨 Clean and responsive user interface
* 🌐 Live deployment on Vercel

---

## 🎯 Project Aim

To build a React application that fetches and displays dynamic data using reusable components and modern frontend tools.

---

## 🧠 What I Learned

* Working with React components and props
* Fetching and displaying dynamic data
* Structuring scalable frontend applications
* Using Vite for fast development
* Deploying projects using Vercel

---

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript (ES6)
* CSS
